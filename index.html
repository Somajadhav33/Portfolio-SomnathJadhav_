Web Development

HTML

HTML is the standard markup language used for creating web pages. It structures content on the web by using a series of elements and tags, such as headings, paragraphs, links, and images.

CSS

CSS is a stylesheet language used to control the presentation and layout of HTML elements. It enables developers to apply styles, such as colors, fonts, and spacing, to web pages, enhancing their visual appeal and user experience. CSS allows for responsive design, ensuring that websites look good on a variety of devices.

Bootstrap

Bootstrap is a popular front-end framework for developing responsive and mobile-first web applications. It provides a collection of pre-designed components and utilities, such as grids, buttons, and navigation bars, that simplify the process of creating visually appealing websites. Bootstrap is built with HTML, CSS, and JavaScript, making it easy to integrate into web projects.

Database Management

SQLite

SQLite is a lightweight, serverless relational database management system (RDBMS) that is widely used for local data storage in applications. It is known for its simplicity and ease of use, making it a popular choice for mobile applications and small to medium-sized projects. SQLite stores data in a single file, allowing for easy management and portability.

DBMS

A Database Management System (DBMS) is software that enables the creation, manipulation, and management of databases. It provides an interface for users to interact with the data and ensures data integrity, security, and efficient retrieval.
